Coding Style Guidelines
======

Table
------

Avoid using hyphened string as table's key:
X "z-index"(confuse with minus operation)
V "zIndex"(recommended) or "z_index"(okey but not good)

- - -

Function
------

Do this,

```
function foo(bar)
  -- do something
end
```
Not like this,

```
function foo ( bar )
    -- do something
end
```
- - -
